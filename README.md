# Sistem Inventaris Barang - Inventra

Aplikasi manajemen stok barang warung berbasis web dengan arsitektur Decoupled (Frontend VueJS & Backend CodeIgniter 4).

## Fitur Utama
- **CRUD Inventaris:** Kelola barang dengan sistem Kategori.
- **RESTful API:** Integrasi penuh antara frontend dan backend.
- **Autentikasi Aman:** Implementasi Token-based Auth (JWT).
- **Statistik Real-time:** Dashboard ringkasan stok dan barang terbaru.

## Teknologi
- **Frontend:** Vue.js (SPA), TailwindCSS, Axios (Interceptors).
- **Backend:** CodeIgniter 4, MySQL, CORS Policy.

## Cara Menjalankan
1. **Backend:** Masuk ke folder `backend-api`, jalankan `php spark serve`.
2. **Frontend:** Buka `frontend-spa/index.html` via Live Server.
3. **Database:** Pastikan import SQL yang tersedia di folder `database/`.
